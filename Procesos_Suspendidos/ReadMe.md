# Practica de simulación de paginación simple



#### Instrucciones

Indica el numero de procesos a crear.

Indica el quantum para el algoritmo round robin.

Una vez indicado los datos anteriores la simulación empezara, se puede interactuar con la simulación con las siguientes teclas:

* E->Interrupción por entrada-salida (pasa a esta de bloqueado).
* W->Error (termina el proceso en ejecución).
* P->Pausa.
* C->Continuar (Solo tiene efecto si anteriormente se pulso P, B, T).
* N->Nuevo proceso.
* B->Tabla de procesos (imprime el BCP de todos los procesos).
* T->Tabla de páginas (imprime la matriz que representa la memoria).
